# Telegram Portfolio Bot

Telegram-бот-портфолио для презентации услуг, кейсов, проектов и контактов через удобное меню.  
Подходит для фрилансера, разработчика, студии или специалиста, который хочет показывать свои работы прямо в Telegram.

## Возможности

### Пользовательская часть
- `/start` и главное меню
- раздел **Обо мне**
- раздел **Контакты**
- раздел **Мои проекты**
- категории и подкатегории проектов
- карточки проектов с:
  - названием
  - описанием
  - фото
  - ссылкой
  - файлами

### Админ-панель
- вход в админку
- создание категорий
- создание подкатегорий
- добавление проектов
- редактирование проектов
- публикация и скрытие проектов
- удаление проектов
- просмотр статистики
- лог действий администратора

## Стек

- Python
- aiogram 2.25.1
- SQLite
- Docker
- Telegram Bot API

## Запуск локалььно

Установить зависимости: pip install -r requirements.txt

Запустить бота: python bot.py

Через Docker. Собрать образ: docker build -t telegram-portfolio-bot . 

Запустить контейнер: docker run --env-file .env telegram-portfolio-bot
